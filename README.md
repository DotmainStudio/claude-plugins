# DotmainStudio Claude Plugins

Personal Claude Code plugin marketplace for DotmainStudio workflows.

## Install

```bash
claude plugin marketplace add dotmainstudio github:DotmainStudio/claude-plugins
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| `relume-figma` | RELUME-KIT Figma guide — component catalog, naming, design tokens |
| `color-scale` | HEX → 0–100 color scale with semantic UI tokens |

## Usage

```bash
# Install a plugin
claude plugin install relume-figma@dotmainstudio
claude plugin install color-scale@dotmainstudio
```
